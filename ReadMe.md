#### 주제

- 데이터베이스를 활용하여 게시판 구축하기

---

#### 개발환경

- IntelliJ
- Window 10
- Java8
- Spring Project: Gradle
- Spring Dependency: H2-Database, Thymeleaf

#### 개발 기간
